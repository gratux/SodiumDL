# SodiumDL
a simple e621.net interface for .NET

## Support
+ .NET Core 3.1
+ .NET 5

## Why?
I've noticed a severe lack of *good* e621.net API wrappers for the .NET ecosystem.
As an avid fan of this image archival website, I set out to create one.

The name of this project is a pun on Monosodium Glutamate (E621), a common flavor enhancer.
